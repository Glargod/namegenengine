# Glargod's Namegen Engine

**v11.0** — A fast, fun, single-file fantasy name generator with old-school procedural soul.

Born on a Tandy 1000 in BASIC 2.0 back in the day.  
"Glargod" and "Vanrengod" were among the very first names this engine ever produced.

Now modernized with better names, real-time generation, favorites, and Spacebar control.

![Demo](https://via.placeholder.com/800x500/0a0a14/00ffcc?text=Glargod%27s+Namegen+Engine+v11.0)

## Features

- Real-time name generation (as fast as you want)
- **Spacebar** to pause/resume — instant control
- Adjustable speed: Slow / Normal / Fast
- Click any name to copy it
- +10 and +50 burst buttons
- ⭐ Favorites list (saved in your browser)
- Rich fantasy name engine with prefixes, suffixes, titles, oddities, and compounds
- Fully offline — just one HTML file

## How to Use

1. Open `index.html` in any browser
2. Click **Start** or press the **Spacebar**
3. Watch the names flow
4. Click any name you like → it copies automatically
5. Double-click the output area to quickly add the last name to favorites

Perfect for D&D sessions, writing, worldbuilding, or just nostalgic vibes.
